✈️ Airlines Operations Dashboard | Power BI Project
📌 Project Title
Airlines Operations Dashboard – Flight Performance, Delay & Cancellation Analysis
📖 Project Overview

A professional Power BI dashboard developed to analyze 48M+ airline operational records, providing executive-level insights into:

Total flight performance
On-time efficiency tracking
Delay reason analysis
Cancellation monitoring
Airport & city-wise traffic trends
Monthly operational performance

This project helps airlines improve operational efficiency, strategic planning, and customer satisfaction through data-driven insights.

🎯 Business Objective
Monitor airline operational performance
Analyze delays and cancellations
Identify major disruption causes
Evaluate airport-wise traffic distribution
Improve scheduling and resource planning
Support business intelligence decision-making
📂 Dataset Details
48M Total Flights
37M On-Time Flights
2M Carrier Delays
858K Cancelled Flights
Key Metrics:
77.88% On-Time Performance
20.05% Delay Rate
1.79% Cancellation Rate
🛠️ Tools & Technologies Used
Power BI – Dashboard development
Excel / CSV – Data source
SQL – Data transformation
Python – Data cleaning & preprocessing
📊 Dashboard Highlights
Key Visuals:
Total Flights by City
Cancellation/Delay Reason Breakdown
Flight Status by Airport
Monthly Flight Trends
Cancelled Flights by Year
🔍 Key Business Insights
Atlanta (ATL) recorded highest flight traffic
Carrier delays were the major disruption factor
Strong 77.88% on-time efficiency achieved
Cancellation rates improved over time
Peak traffic observed during mid-year months
Major hubs dominate airline operations
